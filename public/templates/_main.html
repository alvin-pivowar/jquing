Sample Directives:


local-datetime directive

This directive (which is used to display the date and time above the menu) is an example of a directive that manipulates the DOM using an embedded template.


jqui-draggable directive

This directive demonstrates:
  • initialization
  • ng-model binding to an object with properties
  • one-way attribute binding
  • attribute callbacks


jqui-accordion directive

This directive demonstrates:
  • initialization
  • ng-model binding
  • one-way attribute binding
  • attribute callbaks


jqui-dialog directive

This directive demonstrates:
  • initialization
  • destruction
  • one-way attribute binding
  • two-way attribute binding
